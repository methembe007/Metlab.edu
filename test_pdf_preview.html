Common Issues & Solutions

  • Blank iframe: Check browser console for errors, verify you're logged in as a student
  • 404 Error: Verify the content ID (2) exists and is assigned to your class
  • 403 Forbidden: Make sure you're enrolled in a class that has this content
  • PDF doesn't render: Try a different browser or enable PDF viewer in browser settings
  • X-Frame-Options error: This is expected - the header allows same-origin iframes
