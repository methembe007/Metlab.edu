Need Help?

Where does my child find the link code?

Your child should log into their account, go to Profile Settings, and look for "Parent Account Linking" section. They can generate a new code there.

What if the link code doesn't work?

Link codes expire after 24 hours for security. Ask your child to generate a new code and try again.

Can I link multiple children?

Yes! You can link as many children as needed. Each child will need to provide their own link code.

What information can I see?

Once linked, you can view your child's learning progress, set screen time limits, receive progress notifications, and send encouragement messages.
